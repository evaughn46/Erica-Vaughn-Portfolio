AI-Enhanced Demand Planner — Power Pages + Azure

Self-service tool that aligns Blue Yonder baseline forecasts with evolving planograms by linking to historical performance.

Overview

I built a planner-facing experience on Microsoft Power Pages with Power Apps workflows, leveraging Azure SQL/ADLS Gen2 to store planogram lineage and sales mappings. The tool scales the Blue Yonder baseline using historicals tied to new layouts.

Business Challenge

  • Manual scaling of forecasts to new planograms caused delays and inconsistency.
  • Needed an intuitive, governed, and auditable process for planners.

Solution

  • Power Pages UI + Power Apps orchestration.
  • Historical linkage of planograms in Azure SQL / ADLS.
  • Scaling logic using prior performance to inform shipments.
  • Optional Power BI views for leadership scenario reviews.

Impact

  • Hours of manual effort removed through automation.
  • Improved shipment accuracy and reduced overstock risk.
  • High planner adoption via familiar Microsoft experience.
